# This app is for discord bot.

## How to prepare enviroment.

```command
git clone https://github.com/SQZ777/ntericeapp.git
npm install
node app.js
```
you need add .env file in your local.

the .env key value should provide:

```
discordToken=
twitchAuthorization=
twitchClientId=
mongoHost=
TRNApiKey=
```
